# Summit Webworks Fresh Repo

This folder is a clean, standalone repository copy of the website so you can push it to a brand-new GitHub repository.

## Quick publish steps
1. Create a new empty repo on GitHub (no README).
2. From this folder:
   ```bash
   cd summitwebworks_fresh_repo
   git init
   git add .
   git commit -m "Initial Summit Webworks website"
   git branch -M main
   git remote add origin <YOUR_NEW_REPO_URL>
   git push -u origin main
   ```
